# Threshold

This is simple plugin that allows a threshold filter to be applied to any image. Simply right-click an image in your vault and then press "Apply threshold" in the pop-up menu.

If the file name matches the pre-existing file, it will overwrite it. If a unique name is given, it will create a new file (of matching extension type) in the directory of the edited image.
